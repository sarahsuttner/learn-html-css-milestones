Attributes

The attributes on the link include "href", "id", "class", and "target"


Href is used to link the URL to the page. It's value here is "https://google.com"

Id assigns an identification quality to the link. It's value here is "the-best-link"

Class assigns the link a class value that is attached to a stylesheet. It's value here is "banana"

Target specifies the manner in which the corresponding web page will open. It's value here is "blank"
