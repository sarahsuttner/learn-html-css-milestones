Inline Text Styling

There are many tags that can effect the styling of text on a site. The "em" tag, for example, is used to emphasize a given word, as shown. The "i" tag is used to italicize a word. The "strong" tag creates thicker, more defined text, similarly to how the "b" tag creates a bolder text type. Lastly, the "code" tag shows computer code, as displayed in monospace font.
